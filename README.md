# HR-Data-Analytics-project
HR Data Analytics project analyzing employee data to uncover insights on performance, satisfaction, and attrition using Python, SQL, and Tableau.
This project was completed as part of the Digital Egypt Pioneers Initiative (DEPI) internship program. The goal of the project was to analyze HR data and extract actionable insights to support better decision-making.

# Project Overview
The dataset includes information on 1,470 employees across multiple departments in a U.S.-based company. The analysis focuses on key HR metrics such as demographics, performance, job satisfaction, salary, overtime, and employee attrition.

# Objectives
- Analyze employee behavior and organizational trends  
- Identify factors affecting employee satisfaction and attrition  
- Provide data-driven recommendations to improve retention and performance  

# Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)  
- SQL Server  
- Tableau  
- Excel  

# Dashboards
- Overview Dashboard  
- Employee Dashboard  
- Salary & Overtime Dashboard  
- Performance Dashboard  
- Attrition Dashboard
